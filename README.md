# File Encryption/Decryption Tool

## Описание

Этот проект состоит из двух утилит для шифрования и дешифрования текстовых файлов с использованием алгоритма Fernet (симметричное шифрование). 

- **lock.py** — шифрует файлы
- **unlock.py** — расшифровывает файлы, выполняет их, а затем автоматически повторно шифрует

Особенности `unlock.py`:
- Автоматически изменяет атрибуты файла (скрывает/показывает)
- Выполняет расшифрованный файл (например, Python скрипт)
- После выполнения автоматически повторно шифрует файл
- Восстанавливает скрытые атрибуты

## Требования

- Python 3.x
- Библиотека `cryptography`
- Пользовательский модуль `mylib` (должен содержать функции `aniinp` и `aniprint`)
- Windows (используются команды `attrib` для управления атрибутами файлов)
